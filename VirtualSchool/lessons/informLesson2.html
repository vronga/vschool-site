Урок 2: Переменные и типы данных

Привет! Сегодня мы познакомимся с основами программирования — переменными и типами данных.

Что такое переменная?

Переменная — это как коробка с этикеткой, в которую можно положить какое-то значение и потом использовать это значение в программе.

Типы данных

Разные виды информации — это разные типы данных. Вот основные из них:

  • Числа — например, 10, 3.14
  • Строки — текст, например, "Привет!"
  • Булевы значения — true (истина) или false (ложь)

Как объявлять переменные?

В JavaScript для создания переменной используется ключевое слово let или const:

let возраст = 12;
const имя = "Анна";

Значение можно менять, если переменная объявлена через let, но не если через const.

Пример

Создадим переменную баллы и присвоим ей значение 8:

let баллы = 8;

Позже можно увеличить количество баллов:

баллы = баллы + 2; // Теперь баллы равно 10

Почему это важно?

Переменные позволяют хранить данные и работать с ними в программе — считать, сравнивать, менять.
