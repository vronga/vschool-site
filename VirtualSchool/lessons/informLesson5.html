Урок 5: Алгоритмы с условиями и циклами

В этом уроке мы подробно разберём два важных понятия в алгоритмах — условия и циклы. Они помогают создавать более гибкие и мощные алгоритмы, которые могут принимать решения и повторять действия.

Что такое условие?

Условие — это проверка, которая позволяет алгоритму выбрать, какой путь выполнить дальше. Обычно условие задаётся в виде вопроса, на который можно ответить "да" или "нет".

Например: Если на улице идёт дождь, взять зонт.

В программировании это выглядит как оператор if:

if (дождь) {
  взять зонт;
} else {
  не брать зонт;
}

Что такое цикл?

Цикл — это способ повторять одно и то же действие несколько раз, пока выполняется определённое условие.

Например: Пока не выучил все слова — повторять их.

В программировании есть разные виды циклов, например, while и for. Вот пример цикла while:

while (не выучил все слова) {
  повторять слова;
}

Почему условия и циклы важны?

  • Позволяют алгоритму принимать решения и адаптироваться к разным ситуациям.
  • Дают возможность повторять действия без копирования одного и того же кода много раз.
  • Упрощают и делают алгоритмы более понятными и эффективными.

Пример алгоритма с условием и циклом

Допустим, нам нужно составить алгоритм для проверки пароля и повторного ввода, пока пароль неверный:

  1. Попросить пользователя ввести пароль.
  2. Проверить, правильный ли пароль.
  3. Если пароль неверный, попросить ввести снова.
  4. Если правильный — показать сообщение "Доступ разрешён".

В виде псевдокода это будет:

do {
  запросить пароль;
} while (пароль неверный);

показать "Доступ разрешён";

Задание

Попробуй составить свой алгоритм с условием и циклом для любой повседневной задачи. Например:

  • Проверка наличия школьных принадлежностей перед выходом из дома.
  • Повторение упражнений, пока не будет сделано заданное количество.
  • Проверка, закончился ли заряд телефона, и если да — подключить зарядку.
